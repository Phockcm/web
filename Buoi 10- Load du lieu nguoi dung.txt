Câu 3. Gọi API để Load Dữ Liệu
3.1. Load dữ liệu sản phẩm – Trang Shop
File: app/(shop)/products/page.jsx
Gợi ý:
Chuẩn bị
ProductService
Hàm getProducts() trả về dữ liệu có cấu trúc:
{
  products: [],      // mảng sản phẩm
  currentPage: 1,    // trang hiện tại
  totalPage: 5       // tổng số trang
}
Thực hiện trong ProductsPage
Khai báo state:
const [products, setProducts] = useState([]);
Gọi getProducts() trong useEffect
Lưu dữ liệu vào state products
Render component ProductList Truyền props:
<ProductList products={products} />
Nhiệm vụ của ProductList
Chỉ hiển thị danh sách sản phẩm
Không gọi API
Không xử lý logic dữ liệu
3.2. Load dữ liệu CHI TIẾT SẢN PHẨM
File: app/(shop)/products/[slug]/page.jsx
Gợi ý:
Chuẩn bị
ProductService
Hàm getProductBySlug(slug) trả về dữ liệu:
{
  id: 1,
  slug: "",
  name: "",
  price: 0,
  image: "",
  description: ""
}
Thực hiện trong ProductDetailPage
Lấy slug từ params
Khai báo state:
const [product, setProduct] = useState(null);
Gọi getProductBySlug(slug) trong useEffect
Lưu dữ liệu vào state product
Render component ProductDetail, truyền props:
<ProductDetail product={product} />
Nhiệm vụ của ProductDetail
Chỉ hiển thị chi tiết sản phẩm
Không gọi API
Không xử lý logic dữ liệu

3.3. Load dữ liệu sản phẩm – Trang Admin
File: app/admin/products/page.jsx
Thực hiện
Khai báo state:
const [products, setProducts] = useState([]);
Gọi getProducts() trong useEffect
Lưu dữ liệu vào state products
Render component AdminTable
Truyền props:
<AdminTable products={products} />
Yêu cầu bổ sung
Hiển thị thêm các cột:
Status
Image
Price
Sinh viên tự xử lý giao diện bảng
